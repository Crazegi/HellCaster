Drop wall texture images here (JPG or PNG).

Supported names:
- wall_brick.jpg (or .png/.jpeg)
- wall_concrete.jpg (or .png/.jpeg)
- wall_metal.jpg (or .png/.jpeg)

Recommended size:
- Minimum: 64x64
- Best: square textures like 128x128 or 256x256

Path notes:
- Development: src/HellCaster.App/Textures/
- Packaged release when running release/HellCaster-latest.exe: release/latest/Textures/

If a file is missing, HellCaster falls back to built-in textures.
